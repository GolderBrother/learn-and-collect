1: 先安装firefox51.0.1(32位版本)
2、然后安装vlc3.0.6(vlc-3.0.6-win32 32位版本的亲测可以)，在“工具--》偏好设置---》输入/编解码器---》选中rtp over rtsp”
vlc下载地址：https://vlc-media-player.en.softonic.com/
3、火狐浏览器52以下版本（亲测52.0.1的32位可以，最大选择51版本） 安装完必须把自动更新关闭,否则下次打开就自动更新到最新版的火狐了
4、火狐浏览器各版本的下载地址 http://ftp.mozilla.org/pub/mozilla.org//firefox/releases/
